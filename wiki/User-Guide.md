# User Guide

## Basic Usage

### 1. Starting the Application
```bash
streamlit run app.py
```

### 2. Navigation
The tool consists of six main tabs:
- 🎯 Ballistic Calculator
- 📊 Ammunition Analysis
- 📈 Advanced Metrics
- 🔍 Bullet Visualization
- 🔫 Firearm Compatibility
- ℹ️ Help & Information

### 3. Input Parameters
Located in the sidebar:
- Bullet Weight (grains)
- Muzzle Velocity (fps)
- Caliber selection
- Bullet Type
- Barrel Length

## Feature Guides

### Ballistic Calculator
1. Enter bullet parameters
2. Click "Calculate Ballistics"
3. View results:
   - Ballistic Coefficient
   - Trajectory Analysis
   - Energy Calculations

### 3D Visualization
1. Select bullet type and parameters
2. View interactive 3D model
3. Use mouse to:
   - Rotate view
   - Zoom in/out
   - Reset view (double-click)

### Firearm Compatibility
1. Select caliber
2. View compatible firearms
3. Access detailed specifications
4. Review safety guidelines
